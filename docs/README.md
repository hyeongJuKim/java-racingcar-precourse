## 기능 목록

### racing car game
* 자동차 이름 목록 입력 받기
* 횟수 입력 받기
* 각 차수별 실행 결과 출력
* 우승자 출력(공동 우승자 포함)
* 예외 상황 발생시 에러 문구 출력

### cars
* 자동차 목록 담기

### car
* 자동차 생성
* 이름 등록
* 전진 횟수
